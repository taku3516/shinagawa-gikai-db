# 品川区議会DB

品川区議会の公式情報を基に、住民が読みやすい形に整理したページです。

- **サイト名**: 品川区議会DB
- **対象**: 令和元年〜8年の品川区議会日程・議案・質問・会議録
- **免責**: これは品川区が運営しているものではありません。正確な議決結果・日程・資料は必ず[品川区議会公式サイト](https://gikai.city.shinagawa.tokyo.jp/)を確認してください。

## 機能

- 📅 **年間の流れ** — 定例会・臨時会・月例委員会をカード形式で表示
- 📝 **会議の詳細** — 日付ごとの出来事、公式資料・録画・議事録へのリンク
- 🗳️ **議案・請願・陳情** — 令和4年〜8年は公式ページの掲載分を全件収録
- 💬 **代表質問・一般質問** — 令和4年〜8年は全発言項目を掲載し、取り下げを除いて会議録からの質問・答弁要約を収録
- ↕️ **段階的な開閉** — 会議、資料種別、質問者ごとに情報量を調整
- 📖 **会議録** — 正式版への案内と校正原稿PDF一覧
- 🔗 **参照元** — 品川区議会公式3ドメインのみ

## 技術スタック

- HTML/CSS/JavaScript のみ（フレームワーク・ビルド不要）
- データは `data/*.js` に分離（`window.SHINAGAWA_DB` に登録するスクリプト形式）
- `data/site.js` — 共通設定
- `data/r04.js` / `data/r04-complete.js` — 令和4年の基礎データと全件表示用補足データ
- `data/r05.js` / `data/r05-complete.js` — 令和5年の基礎データと全件表示用補足データ
- `data/r06.js` / `data/r06-complete.js` — 令和6年の基礎データと全件表示用補足データ
- `data/r07.js` / `data/r07-complete.js` — 令和7年の基礎データと全件表示用補足データ
- `data/r08.js` — 令和8年のデータ（会議・議案・請願・質問・会議録）
- `data/r08-complete.js` — 令和8年の議案・請願・陳情・質問を全件表示する補足データ

令和元年〜3年の全件化と会議録要約については、[実装準備・監査手順](docs/history-expansion.md)に公式件数、収集方法、検証・生成手順をまとめています。

スクリプト形式のため、**サーバーなしで `index.html` をダブルクリックするだけで動きます**（GitHub Pages等での公開もそのまま可能）。

## 使い方

### ローカルで開く

`index.html` をダブルクリックしてブラウザで開くだけです。

HTTPサーバー経由で確認したい場合は次でも開けます：

```bash
cd shinagawa-gikai-db
python3 -m http.server 8000
```

### 過去年・未来年のデータを追加

1. `data/r07.js` など新しい年のファイルを作成（`data/r08.js` と同じ形式。冒頭の登録先を `window.SHINAGAWA_DB.years["r07"]` にする）
2. `data/site.js` の `years` 配列で該当年の `available` を `true` にする

例：
```json
{
  "id": "r07",
  "label": "令和7年",
  "file": "data/r07.js",
  "available": true
}
```

## データ設計

### リンクの色分け

- **青（official）** — 公式資料・日程ページ
- **赤（video）** — 録画中継ページ
- **緑（minutes / minutesDraft）** — 会議録検索 or 校正原稿PDF

### リンク型の説明

| 型 | 用途 |
|---|---|
| `official` | 公式資料・日程へのリンク |
| `video` | インターネット中継の録画へのリンク |
| `minutes` | 会議録検索システム（正式版）へのリンク |
| `minutesDraft` | 校正原稿PDF（※正式版公開後に削除される） |

校正原稿PDF型のリンクが含まれる場合、自動で「削除される場合の対策」の案内が表示されます。
